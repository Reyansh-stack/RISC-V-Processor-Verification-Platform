# RISC-V-Processor-Verification-Platform
Industry-style RISC-V processor design and verification platform built from scratch using Verilog, SystemVerilog, and UVM.

## Overview

This repository documents my journey of designing and verifying a 32-bit RISC-V processor using industry-standard methodologies.

The project begins with digital logic fundamentals and progressively builds toward a complete RTL implementation and a professional verification environment similar to those used in semiconductor companies such as NVIDIA, Intel, AMD, Qualcomm, Synopsys, and Cadence.

## Goals

- Learn Digital Design
- Master Verilog RTL Design
- Learn SystemVerilog
- Build reusable verification components
- Learn Assertions (SVA)
- Learn Functional Coverage
- Learn Constrained Random Verification
- Learn UVM
- Build a complete RISC-V Processor Verification Platform

## Repository Structure
